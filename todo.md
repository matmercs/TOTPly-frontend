Must-have:
- Регистрация и аутентификация пользователей
- Сессии (таймаут, JWT токен)
- Базовая безопасность (HTTPS, CORS)
- API для фронта
- Добавление Base32-секрета
- Безопасное хранение Base32-секрета (шифрование)
- Генерация TOTP-кодов(RFC6238)
- Просмотр списка TOTP-аккаунтов
- Редактирование и удаление записей

Nice-to-have:
- Интеграция с Vault(HashiCorp) для хранения секретов
- Экспорт/импорт TOTP-секретов в зашифрованном виде
- Инструменты для сбора статистики, логирования, аудита событий, мониторинга состояния системы
- CI/CD
